1. This set is a simple feature designed to conveniently modify the decibels of a WAV file.
2. This set contains two demo audio files. Use this to make it simple.
3. Two demo audio were licensed under 'Creative Commons 0' from freesound.org .

1. 이 에셋은 WAV 파일의 데시벨을 편리하게 수정하기위해 만들어진 간단한 기능입니다.
2. 이 에셋에는 두개의 데모 오디오 파일이 있습니다. 이것을 이용하여 간단하게 사용해보세요.
3. 두개의 데모 오디오는 freesound.org에서 'Creative Commons 0'라이센스로 받았습니다.

1. Open the Unity Editor.
2. Go to the "Tools" menu and select "Bonnate" > "WAV Easy Volume Editor."
3. In the editor window:
- Choose a WAV audio clip from the Unity Project window (Only .wav files are selectable).
- Set the target decibel (dB) level you want to achieve.
- Click the "Process and Replace" button to adjust the audio clip in place, creating a backup of the original.
- Click the "Process and Export" button to adjust and save the processed audio clip to a new file.
- Use the "Listen" button to preview audio at the specified dB level.
4. The script will display audio clip details, including maximum dB, sample rate, channels, and length.

1. Unity Editor(유니티 편집기)를 엽니다.
2. 도구 메뉴로 이동하여 "Bonnate" > "WAV Easy Volume Editor"를 선택합니다
3. 편집기 창에서:
- Unity Project(유니티 프로젝트) 창에서 WAV 오디오 클립을 선택합니다(.wav 파일만 선택 가능).
- 달성하려는 목표 데시벨(dB) 레벨을 설정합니다.
- "Process and Replace(프로세스 및 교체)" 버튼을 클릭하여 오디오 클립을 제자리에서 조정하여 원본의 백업을 만듭니다.
- 처리된 오디오 클립을 조정하고 새 파일에 저장하려면 "Process and Export" 버튼을 클릭합니다.
- 지정된 dB 레벨에서 오디오를 미리 들으려면 "Listen" 버튼을 사용합니다.
4. 스크립트에는 최대 dB, 샘플 속도, 채널 및 길이 등 오디오 클립 세부 정보가 표시됩니다.
